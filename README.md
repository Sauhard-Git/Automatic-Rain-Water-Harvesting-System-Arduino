# 📖 Overview

The **Automatic Rain Water Harvesting System** is an Arduino-based automation project designed to collect rainwater automatically whenever rainfall is detected.

The system uses a **Rain Sensor** to detect the presence of rain. Once rain is detected, the **Arduino Uno** activates a **Servo Motor**, which rotates a collection mechanism to direct rainwater into a storage container. When rainfall stops, the servo returns to its original position, preventing unnecessary water collection.

This project demonstrates the practical implementation of **Embedded Systems**, **Sensor Interfacing**, and **Automation** for sustainable water conservation.

# ⚙️ Working Principle

```text
System Starts
      │
      ▼
Rain Sensor Detects Rain
      │
      ▼
Rain Detected?
   ┌───────────────┐
   │               │
 No              Yes
   │               │
   ▼               ▼
Servo Closed   Servo Opens
   │               │
   ▼               ▼
Waiting      Water Collection
      │
      ▼
Rain Stops
      │
      ▼
Servo Returns to Initial Position
```
